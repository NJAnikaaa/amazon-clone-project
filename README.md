## Challenges I faced

### Problem 1: Navigation Bar Alignment
The navigation items weren't aligning properly at first. I learned about `justify-content: space-evenly` and `align-items: center` in Flexbox to fix this!

### Problem 2: Product Card Layout
Getting those 4 mini images inside each product card was tricky. I solved it using CSS Grid with `grid-template-columns: 3fr 3fr`.

### Problem 3: Images Not Loading
I had to make sure all image paths were correct. Learned about relative paths like `/images/filename.jpg`.

### Problem 4: Footer Styling
The footer had multiple sections with different background colors. I split it into separate divs and styled each one.
